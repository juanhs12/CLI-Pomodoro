## Flowmodoro Mode
The Flowmodoro mode offers a flexible approach to task management, allowing users to define a maximum work duration and work towards it at their own pace.

- **Non-blocking Progress:** The timer tracks your total work progress towards the goal in real-time.
- **Intelligent Pause (Ctrl+C):** If you need to stop to take a break, simply press `Ctrl+C`. The application will calculate the work done since the last pause (or start) and suggest a break equivalent to 20% of that time.
- **Auto-Resume:** After the calculated break period completes, the application will automatically resume the Flowmodoro timer, continuing from where you left off until the total work goal is achieved.
- **Data Logging:** As with standard Pomodoro sessions, session data is recorded to your CSV file upon completion or interruption.
